# Article Comparison

Comparing article overlap with pubmed searches

## Overview
This repository contains tools and data for analyzing overlaps between PubMed article searches across different queries or datasets.

## Getting Started
1. Upload your PubMed article files to this repository
2. Use the comparison tools to identify overlapping articles
3. Generate reports on article overlap statistics

## File Structure
- `/data/` - Store your PubMed article CSV/JSON files here
- `/scripts/` - Python scripts for comparison analysis
- `/results/` - Generated comparison reports

## Contributing
Add your article files and comparison analyses to help build this resource.
